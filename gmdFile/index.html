If a .gmd file doesn't work

Change the ".gmd" in the filename to ".xml" and open the file in a text editor.

Look for an s tag ("<s>") that contains the level string. e.g. <s>H4sIAAAAAAAAC619y64sO47dD5...</s>

It's usually bigger than anything else in the gmd file, so that's how you'll know.

Sometimes you may have to add H4sIAAAAAAAA to the beginning of the level string

Now paste the level code in the textbox of the previous page
